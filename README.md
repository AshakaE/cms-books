# React 

> A library built with react, and redux used for managing the state of the application.

![](./src/utils/screenshot.png)

## Built With

- React.js
- TailwindCSS

## [Bookstore CMS](https://cms-books.herokuapp.com/)

[React]()

## Getting Started

- `git clone git@github.com:AshakaE/cms-books.git`
- `cd cms-books`
- `npm install`
- `npm start`

### Prerequisites

Node.js
